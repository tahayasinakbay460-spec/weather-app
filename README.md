 **Weather App (PyQt5)**

A simple yet modern PyQt5 weather application that displays real-time weather data using the OpenWeatherMap API.
Includes multi-language support (English & Turkish) powered by Google Translate.

** Features***
--
 -Real-time weather data (temperature, humidity, pressure, wind speed)

 -Weather condition icons and short description

 -5-day forecast support

 -Language switcher (English ↔ Türkçe)

-Basic caching system to reduce API calls

**Libraries Used**

-PyQt5 – GUI framework

-requests – HTTP requests to API

-googletrans – automatic translation

-datetime – for date and time formatting

**How to Run**

-Clone this repository:

-git clone https://github.com/tahaysin-akbay/weather-app.git

-Navigate to the project folder:

-cd weather-app

-Install the required libraries:

-pip install PyQt5 requests googletrans==4.0.0-rc1

-Run the app:

-python main.py
